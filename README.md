# vue-covid-19

## 使用到的技术

1. [Axios 网络请求](http://www.axios-js.com/)
2. [Echarts 数据可视化](https://echarts.apache.org/zh/index.html)
3. [Swiper 轮播图](https://www.swiper.com.cn/)
4. [Vue2 基础知识](https://cn.vuejs.org/v2/guide/)
5. [Vue-cli 脚手架](https://cli.vuejs.org/zh/)
6. [Vue-router 路由](https://router.vuejs.org/zh/)
7. [移动端 UI 框架：Vant](https://vant-contrib.gitee.io/vant/#/zh-CN/)
8. Proxy 代理跨域
